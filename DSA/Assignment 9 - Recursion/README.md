Given a number n, find the sum of the first natural numbers.

**Example 1:**

Input: n = 3

Output: 6

**Example 2:**

Input : 5

Output : 15

## Solution

```javascript
function sum(n) {
	if (n == 1) return 1;
	return n + sum(n - 1);
}
```

Given a positive integer, N. Find the factorial of N.

**Example 1:**

Input: N = 5

Output: 120

**Example 2:**

Input: N = 4

Output: 24

## Solution

```javascript
function factorial(n) {
	if (n == 1 || n == 0) return 1;
	return n * factorial(n - 1);
}
```
